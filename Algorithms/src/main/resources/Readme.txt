通过封装一个最大堆这个算法，学到的知识点
    1.写一个Iterator,发现官方的源码Iterator接口，竟然实现了方法。感觉很奇怪，问什么接口中，还要实现方法，接口中
      是不能有方法的实现的啊，查阅资料发现，jdk8以后，可以在接口中有实现的方法，不过方法的实现前面必须加上default
      关键字。其实这是java的又一次进步，可以在接口中有实现方法，这样可以进一步增加java的灵活性了，也使我们开发降低
      了工作性。http://blog.csdn.net/hbsyqjj/article/details/42567671,这是一篇java8关于接口中有实现方法
      的文章。
